Included are a number of mathematical approaches to estimate a single reflectance spectrum from a single RGB triplet.

Written in Octave, a free and open source Matlab alternative

Code by Scott Burns from the following publications: 

	Burns, Scott (2015). Generating Reflectance Curves from RGB Triplets. arXiv:1710.05732. URL: http://scottburns.us/reflectance-curves-from-srgb/

	Burns, Scott (2015). Subtractive Color Mixture Computation. arXiv:1710:06364. URL: http://scottburns.us/subtractive-color-computation/

TO USE:
1. Download Octave
2. Open Octave Command Line Interface
3. Run cd S:\GitProjects\antcolor\antcolor\SpectraFromRGB
4. Run METHOD([RED,GREEN,BLUE] i.e. LLSS([108,67,35])
5. Copy output column to Excel or whatnot for analysis, this is the spectra
